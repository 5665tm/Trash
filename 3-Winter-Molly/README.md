Бот предназначенный для торговли на бирже криптовалюты BTC-E. В данном публичном репозитории представлена лишь альфа версия (конкретно 0.2s, от 3 дек. 2013г) в возможности которой входят следующие функции:

* Получение информации о текущем балансе на каждом из кошельков
* Получение информации о курсах всех валют
* Подсчет общего баланса в рублевом эквиваленте

В программе используется API сайта btc-e.com, ознакомиться с которым можно здесь - https://btc-e.com/api/documentation
Для полноценного использования программы необходимо изменить APIKEY и SECRET на соответствующие вашему аккаунту.
